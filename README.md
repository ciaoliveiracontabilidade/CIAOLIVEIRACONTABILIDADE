# Sobre o projeto

O projeto foi desenvolvido para representar digitalmente a identidade da **CIAOLIVEIRA CONTABILIDADE**, oferecendo um ambiente moderno, elegante e responsivo para atendimento de clientes.

O site possui integração direta com WhatsApp, organização por setores e identificação automática de horário de funcionamento.

---

# Funcionalidades

- Interface moderna e responsiva
- Integração com WhatsApp
- Identificação automática de atendimento online
- Animações suaves
- Layout corporativo premium
- Compatível com dispositivos móveis
- Estrutura separada em HTML, CSS e JavaScript

# Horário automático

O sistema identifica automaticamente:

- dias úteis
- horário comercial
- status de atendimento

Exibindo:

```txt
Atendimento disponível agora
```

ou

```txt
Fora do horário de atendimento
```
